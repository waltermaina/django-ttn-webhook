About

RAK1906 WisBlock Environmental Sensor and Django Project

I created this project so that it could help me put into practice skills that I had learnt about: Heroku, Django, Git, Bootstrap, REST, Python, RAK1906 WisBlock Environmental Sensor, LoRaWAN, Arduino and MongoDB.

In this project, temperature, humidity, air pressure and air quality are measured using RAK1906 WisBlock Environmental Sensor which is connected to an RAK4630 module. The RAK4630 module sends the data to The Things Network (TTN) which forwards the data to a Django rest api which validates and stores the data in a MongoDB database. A front end app in the Django project then displays the data using four graphs: for temperature, humidity, air pressure and air quality.
